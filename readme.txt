Places a simple "Search" box to the APRS.fi website as a widget on your site for searching Amateur Radio APRS beacons.

== Description ==

<p>This widget places a search box to the APRS.fi website on your site.  Use this search box to search the call sign 
of another Amateur Radio (Ham) operator on <a href="http://www.aprs.fi">APRS.fi</a>.</p>

<p>The following aspects are customizeable:
<ul>
<li>Widget title</li>
<li>Initial search value</li>
<li>Search button text</li>
</ul>
The following aspects are not *yet* customizeable:
<ul>
<li>The width of the text input</li>
<li>Zoom level</li>
</ul>
</p>
<p>Pressing the search button pops open a new window with the search results.</p>

== Installation ==

1. Upload `aprs-fi-search-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to one of your theme's widget areas.
